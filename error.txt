1. SyntexError - ошибка синтаксиса при прописании кода.
Например:
N = input['Введите число: ') - SyntaxError, "[" ')'
2. NameError - ошибка, которая возникает когда используется переменная, функция или модуль,
которые не заданы ранее при прописании кода.
Например:
print(b)
NameError: name 'b' is not defined - переманная b не задана ранее

3.IndentationError - oшибка, которая возникает когда не соблюдены отступы в коде.
Например:
A = int(input())
if A > 5
print("OK")  - не соблюден отступ

4. TypeError - ошибка, которая возникает при операции с несоответствующим объектом.
Например:
a = 1
b = "hello"
print(a+b)
TypeError: unsupported operand type(s) for +: 'int' and 'str'

5.ValueError - ошибка, которая возникает при передаче аргумента с не поддерживающим значением.
Например:
int("Hello world")
